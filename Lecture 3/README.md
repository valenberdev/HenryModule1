# 🎬 Recomendaciones de Anime, Películas y Series

## 📖 Descripción
Este proyecto es una página web simple creada con HTML y CSS donde comparto mis recomendaciones personales de animes, películas y series. El objetivo es practicar desarrollo web mientras construyo un sitio visualmente atractivo con estilo moderno inspirado en plataformas de streaming.

---

## 📂 Contenido
Actualmente el proyecto incluye:

- 🧱 Estructura HTML semántica (`header`, `nav`, `section`, `footer`)
- 🎨 Estilos con CSS moderno (reset + variables)
- 🎴 Cards interactivas estilo Netflix
- 📱 Diseño responsive con CSS Grid
- ✨ Animaciones hover en imágenes
- 🔤 Tipografía personalizada con Google Fonts (Poppins + Inter)

---

## 🚀 Cómo usar
1. Clonar o descargar el repositorio  
2. Abrir el archivo `index.html` en el navegador  
3. Navegar entre secciones usando el menú (Animes, Películas, Series)  
4. Pasar el mouse sobre las cards para ver más información
